# Cruise Ship Management System

A university Information Systems project designed to manage cruise ship operations, customer reservations, rooms, activities, employees, and billing.

## Project Contents
- Documentation
- Microsoft Access Database
- Power BI Dashboard

## Features
- Customer Management
- Reservation Management
- Room Management
- Employee Management
- Billing System
- Reports and Queries
- Dashboard and Data Visualization

## Technologies Used
- Microsoft Access
- Power BI
- ERD Design
- Database Queries and Reports
