# Presentation

This folder contains the presentations developed for the Cruise Ship Management System project.

## Contents

* Database Project Presentation
* Software Engineering Presentation
* Project Demonstration Materials

## Description

The presentations provide an overview of the Cruise Ship Management System, including project objectives, requirements analysis, system architecture, UML and software engineering artifacts, database design, implementation details, business intelligence reporting, GUI design, and project outcomes.

These presentations were used to demonstrate the project during academic evaluations and project presentations.
