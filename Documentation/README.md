# Documentation

This folder contains the main documentation and supporting materials for the Cruise Ship Management System project.

## Contents

* Cruise Ship Project Documentation
* Cruise Ship Case Study
* Software Requirements Specification (SRS)
* SQL Documentation
* Database Mapping Documentation
* Project Reports and Supporting Documents

## Description

The documents provide a comprehensive overview of the project, including system requirements, project objectives, business rules, functional and non-functional requirements, use cases, database design, SQL implementation, and overall system analysis.

These materials were developed throughout the planning, analysis, design, database development, and documentation phases of the project, serving as a complete reference for the Cruise Ship Management System.
